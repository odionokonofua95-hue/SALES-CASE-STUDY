## üìê 3. Metrics Developed
3.1 Daily Sales Price per Unit

Calculated as:

Sales √∑ Quantity Sold

3.2 Average Unit Sales Price

The mean of daily unit price values across the dataset.

3.3 Daily % Gross Profit
Gross Profit = Sales ‚Äì Cost of Sales  
Gross Profit % = (Gross Profit √∑ Sales) √ó 100

3.4 Daily % Gross Profit per Unit
Gross Profit per Unit = (Sales ‚Äì Cost of Sales) √∑ Quantity Sold  
% GP per Unit = (Gross Profit per Unit √∑ Unit Price) √ó 100
